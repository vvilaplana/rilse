# RILSE - Roland Integra 7 Linux Sound Editor

RILSE is a **Linux desktop application** for the **Roland Integra-7** sound module.

I wrote this because, despite finding many applications that allow editing studio sets and part parameters, I couldn't find any that let me change the order of studio sets or parts or copy&paste them.

## Requirements
* It only works on Linux x86-64
* Wayland
* gtkmm 4
* libasound2

## Current capabilities
* Studio sets: Copy & Paste, Change order, Edit name, Initialize

## Work in progress
* Part maintenance (Copy&Paste, Change order, Init, etc)
* Part parameter editing
* Sampling capabilities (WAV to FCE-DPCM conversion)
